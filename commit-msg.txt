  PS popup spawn loop when worker handshake fails.

  minimal snapshot so status server stays alive without worker.

  Constructor only assigns if non-null. start() returns early if null.
  All five poll methods guard with if (this.ipc === null) return.

src/tray/lifecycle.ts: Removed as-never casts now that types are
  correct. Worker failure is fully non-fatal — context engine, sniper,
  cognitive load, status server all boot and run without it.
